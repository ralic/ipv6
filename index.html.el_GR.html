Δοκιμάστε την IPv6 συνδεσιμότητά σας. (Επανάληψη)

Απαιτείται JavaScript

Η JavaScript και η χρήση cross-site scripts είναι προαπαιτούμενα για την πραγματοποίηση δοκιμών από αυτή τη σελίδα.

Η παραμονή αυτού του μηνύματος σημαίνει ότι η JavaScript έχει απενεργοποιηθεί, είτε από κάποιο plugin/extension του φυλλομετρητή σας, ή στις ρυθμίσεις του.

Χρησιμοποιείτε NoScript?

Αν χρησιμοποιείτε αυτό το FireFox add-on, τότε θα πρέπει να προβείτε στο "Temporarily allow all this page". Θα πρέπει να το επαναλάβετε αυτό δύο φορές ώστε όλα να λειτουργούν σωστά. Εναλλακτικά, απενεργοποιείστε ολοκληρωτικά το NoScript ώσπου να ολοκληρώσετε τη δουλειά σας σε αυτή τη σελίδα.

Εναλλακτικά μπορείτε να επιλέξετε να δείτε την απλή δοκιμή, η οποία παρουσιάζει εν συντομία επιτυχίες/αποτυχίες για το IPv4, IPv4+IPv6 και IPv6. Ωστόσο αυτές οι πληροφορίες δεν θα είναι πολύ διαγνωστικές.

Πώς λειτουργεί αυτή η δοκιμή: Ο φυλλομετρητής σας θα βαλθεί να προσπελάσει μια σειρά από URLs. Ο συνδυασμός των επιτυχιών και αποτυχιών θα δείξει κατά πόσο έτοιμοι είστε για το όταν οι σελίδες θα προσφέρονται μέσω IPV6.

Κάντε κλικ για να δείτε Αποτελέσματα Δοκιμών

Δοκιμή με IPv4 DNS εγγραφή
εκκρεμές
-
Φέρει ένα αντικείμενο το οποίο έχει απλώς μια Α εγγραφή στο DNS. Αυτό αναμένεται να χρησιμοποιεί IPv4. Χρήστες που χρησιμοποιούν μόνο IPv6 θα μπορούσαν να κάνουν χρήση αυτού, αν ο πάροχός τους προσφέρει μια NAT64/DNS64 ή proxy λύση.
Δοκιμή με IPv6 DNS εγγραφή
εκκρεμές
-
Φέρει ένα αντικείμενο το οποίο έχει απλώς μια AAAA εγγραφή στο DNS. Αυτό αναμένεται να χρησιμοποιεί IPv6. Χρήστες χωρίς IPv6 θα δουν να αποτύχει. Όσο αποτυγχάνει γρήγορα θα είναι εντάξει - για την ώρα.
Δοκιμή με Dual Stack DNS εγγραφή
εκκρεμές
-
Αυτή είναι η πιο σημαντική δοκιμή, η οποία επιβεβαιώνει ότι ο φυλλομετρητής σας μπορεί να προσπελάσει μια σελίδα που προσφέρεται αμφότερα μέσω IPv4 και IPv6. Ένας υπολογιστής χωρίς IPv6 θα πρέπει να συνδεθεί επιτυχώς (μέσω IPv4).

Αν αυτή η δοκιμή αποτύχει ή λήξει την περίοδο timeout, θα αντιμετωπίσετε προβλήματα τη στιγμή που οι σελίδες προσφερθούν μέσω IPv6.
Δοκιμή για Dual Stack DNS και μεγάλο πακέτο
εκκρεμές
-
Επιβεβαιώνει ότι μπορείτε να συνδεθείτε σε ένα dual-stack server (όπως η ds δοκιμή) και ότι μπορείτε να στέλνετε/λαμβάνετε μεγάλα πακέτα δια της σύνδεσης αυτής.
Δοκιμή IPv4 χωρίς DNS
εκκρεμές
-
Θα γίνει προσπάθεια σύνδεσης μέσω της ίδιας της IPv4 διεύθυνσης. Αυτό θα πρέπει να λειτουργεί για τους περισσότερους, εκτός όσων χρησιμοποιούν μόνο IPv6. Αν η πρώτη δοκιμή πέτυχε και η εν λόγω αποτύχει, ενδεικνύεται ότι ο πάροχός σας χρησιμοποιεί NAT64/DNS64· θα πρέπει να χρησιμοποιείτε hostnames αντί IP διευθύνσεων.
Δοκιμή IPv6 χωρίς DNS
εκκρεμές
-
Θα γίνει προσπάθεια σύνδεσης μέσω της ίδιας της IPv6 διεύθυνσης. Σκοπός είναι ο διαχωρισμός συνδεσιμότητας μέσω IPv6, χώρια της δυνατότητας απόκτησης μέσω DNS. Ένας άλλος σκοπός είναι ο έλεγχος αν έχετε ενεργοποιήσει το Teredo· μερικά συστήματα δύνανται μόνο Teredo να χρησιμοποιήσουν, όταν η IPv6 διεύθυνση χρησιμοποιείται στο URL.
Δοκιμή μεγάλου πακέτου IPv6
εκκρεμές
-
Επαληθεύει τη λειτουργία αιτημάτων IPv6 για μεγάλα πακέτα. Αν αυτή η δοκιμή παρουσιάσει timeout, ενώ οι άλλες IPv6 δοκιμές ολοκληρωθούν επιτυχώς, τότε ίσως να πρόκειται για PMTUD θέματα, με πιθανή συμμετοχή των IP tunnels. Ξανά ελέγξτε και βεβαιωθείτε ότι τα ICMPv6 Type 2 ("Packet Too Big") μηνύματα επιτρέπονται από το firewall σας.
Δοκιμή αν ο DNS server του ISP σας χρησιμοποιεί IPv6
εκκρεμές
-
(This is bonus credit)
Αυτή η δοκιμή είναι για τον resolver του ISP σας (όχι για τον υπολογιστή σας). Η επιτυχία αυτής της δοκιμής σημαίνει ότι ο DNS εξυπηρετητής (συνήθως προσφερόμενος από τον ISP σας) δύναται να προσπελάσει IPv6-only DNS authoritative servers στον Internet. Κάτι το οποίο δεν θεωρείται κρίσιμο (για την ώρα) για την προσπέλαση σελίδων μέσω IPv6.
Βρείτε Πάροχο Υπηρεσιών IPv4
εκκρεμές
-
Επιχειρεί να προσδιορίσει τον πάροχο της IPv4 σύνδεσής σας στο Internet. Μπορεί να τον εμφανίσει με διαφορετική επωνυμία από εκείνη που χρησιμοποιεί στην εγχώρια αγορά· ή να αναφέρει μια προηγούμενη επωνυμία που ο πάροχος χρησιμοποιούσε. Η επωνυμία που παρατίθεται εδώ είναι εκείνη με την οποία ο πάροχος είναι γνωστός στην κοινότητα των διαχειριστών δικτύου.
Βρείτε Πάροχο Υπηρεσιών IPv6
εκκρεμές
-
Επιχειρεί να προσδιορίσει τον πάροχο της IPv6 σύνδεσής σας στο Internet. Αν η επωνυμία αυτή δεν ταιριάζει με εκείνη της IPv4 σύνδεσης, τότε πιθανόν χρησιμοποιείτε κάποιο tunnel, ή κάποιον άλλο πάροχο για το IPv6.
Δοκιμή για ελατωματικό DNS
εκκρεμές
-
Queries for a malformed AAAA record. Some routers mishandle these as "A" records, and only keep the first 32 bits. We want this test to fail to connect.

Αν η περίληψη παρουσιάζει προβλήματα, μπορείτε (εσείς ή η τεχνική σας υποστήριξη) να χρησιμοποιείτε τις παραπάνω πληροφορίες για τη διάγνωσή τους. Αριστερά φαίνονται τα urls των δοκιμών και τα αποτελέσματά τους, ενώ δεξιά παρατίθεται η σχετική περιγραφή του τι εξετάζεται.

Μετά την εκτέλεση των δοκιμών, η σελίδα της περίληψης προβαίνει στην εξέταση των αποτελεσμάτων. Αν η περίληψη μοιάζει να μη έχει νόημα (λαμβάνοντας υπόψιν τα συμπτώματα που αναφέρονται παραπάνω), ή χρειάζεστε περισσότερη βοήθεια, μη διστάσετε να επικοινωνήσετε μαζί μας.
